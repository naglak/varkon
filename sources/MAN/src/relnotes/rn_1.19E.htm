Release notes for Varkon 1.19E

svn#121

Bugfixes related to circle/circle intersects and section area calculation. Support for new MBS function EIGASYM().

Varkon 1.19E released.

Varkon 1.19E is functionally the same as Varkon 1.19D but updated to be built with the latest version of the Gnu compiler and the latest C-, X11-, OpenGL- and Tiff- libraries etc. This version of Varkon is built on a system with Ubuntu 12.04 using the KDE development tool.
